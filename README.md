# BudgetLUT

Discord bot written in Haskell for uploading, managing, and applying LUTs* to images. (*Not a true LUT system, simply maps the colors of an input to the closest color in the LUT.) 

## Commands

```
!help
!lut help
!lut add <name> <url>
!lut rename <code>
!lut delete <code>
!lut view <code>
!lut list
!lut apply <code> <url>
```
